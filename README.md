## Usage

mdbookの編集については以下の通り。

1. mainブランチをcloneしてrootディレクトリで以下を実行

```bash
# Run "mdbook serve"
docker-compose up
```

2. `src`内の`md`ファイルを編集

3. `main`へコミット→プッシュ

4. GitHub Actionが走るので`gh-pages`にページが公開される（更新される）
